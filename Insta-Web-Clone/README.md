# 📸 Instagram Clone (React)

A responsive Instagram clone built using React, featuring authentication, routing, and a modern UI.

## 🚀 Features

* User Signup & Login (localStorage based)
* Protected Routes
* Home Feed UI (Instagram-style)
* Elegant Authentication UI
* Responsive Design

## 🛠️ Tech Stack

* React (Vite)
* React Router
* Context API
* LocalStorage

## 📂 Folder Structure

src/

* components/
* pages/
* context/
* services/

## ⚙️ Installation

```bash
git clone <your-repo-link>
cd instagram-clone-react
npm install
npm run dev
```
